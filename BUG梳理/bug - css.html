      # 关于前端编码问题的BUG

      记得在牛客网刷题的是时候碰到了离谱的bug，就是写个圣诞树的大学题目那种，就类似如这种
      最后用例是通过了，但结果显示树是歪的，加了个空格就向右歪，不加向左歪，想到是不是可能是因为字体编码问题后，加上字符种类限制为微软官方字体，即修复成功。因为他们所占用的字节不同

      # css sytle和标签优先级问题

      css写下面style里面不正常 写在上面标签里面就正常的问题，写一个内边距在下面style写死活不正常但有效果，试了半小时，同事告诉我写在上面就正常了。一试果然，问了同事他也不知道是什么原因。
      我都怀疑是前同事在项目里面埋雷。又试了半小时，一个个css看加百度。发现是css优先级问题，在标签里面写CSS样式会比在style标签中写的优先级更高，因此可能会覆盖样式表中的一些样式。
